# network-controller-for-elite-dangerous
This is a Pyhton script that opens a port at local network and hosts a website. You can open that and assign controls to Elite Dangerous.

How to use:
1.Download script anywhere you want.
2.Run it.
3.Connect to localhost:5000 from any browser.
4.Assign controls at Elite Dangerous.
